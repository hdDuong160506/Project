Các bước chạy demo map-route-map:
B1. Cài python, cài các thư viện liên quan:os, json, requests, flask, flask_CORS (tự tải phiên bản mới nhất)
B2. Chạy file server.py bằng 1 trong các cách sau:
- Dùng code runner trên VSCode
- Mở terminal, truy cập đến thư mục map-route-web và gõ "python server.py"

B3. Mở trình duyệt và chạy file html trên "http://127.0.0:5000"
