1. Mở file app.py lên -> Chọn terminal -> New terminal 
2. Tạo môi trường ảo bằng cách nhập vào terminal: python -m venv .venv
3. Kich hoạt môi trường ảo .venv bằng cách: .venv\Scripts\activate  
4. Cài thư viện: pip install -r requirements.txt
5. Chạy file app.py
